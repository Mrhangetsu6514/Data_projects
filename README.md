## 📂 Project Directory

### 🏗 [n8n - Automated Property Pipeline](./n8n)
**Focus:** *Infrastructure-as-Code, DevOps, & API Integration*
* **The Problem:** Fragmented real estate lead management for the Israeli rental market.
* **The Solution:** A containerized (Docker) environment running an n8n automation engine and a PostgreSQL 16 database.
* **Tech:** Docker Compose, PostgreSQL, n8n, Webhooks, SQL.
* **Outcome:** Created a resilient, self-hosted REST API that ingests leads in real-time with 0% data loss during system restarts.

### 🌿 [Plant Disease Detection](./Plant_desease_detection)
**Focus:** *Deep Learning, Computer Vision, & Image Classification*
* **The Problem:** Identifying agricultural diseases manually is slow and error-prone.
* **The Solution:** A custom Convolutional Neural Network (CNN) trained to identify diseases in plants and citrus fruits.
* **Tech:** Python, TensorFlow/Keras, Google Colab, Computer Vision.
* **Outcome:** Developed an end-to-end machine learning pipeline from data preprocessing and augmentation to model fine-tuning and evaluation.
